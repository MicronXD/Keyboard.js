Keyboard.js

A small lib which provides a simple, sensible api for keyboard mapping

View project on GitHub

API

Keyboard.map

Keyboard.map(keycode, downFn, upFn /**, preventDefault **/);

keycode - The keycode to bind to. downFn - The function to be called when the key is pressed. upFn - The function to be called when the key is released. preventDefault - if set to true, the keydown and keyup events will have preventDefault called on them.

Usage
Keyboard.map(
  32,
  function(e){ console.log('the spacebar has been pressed!'); }
);
Keyboard.map(
  32,
  function(e){ console.log('the spacebar has been pressed!'); },
  function(e){ console.log('the spacebar has been released!'); }
);

Keyboard.unmap

Keyboard.map(keycode);

keycode - The keycode to unbind.

Usage
Keyboard.unmap( 32 );
Keyboard.unmap( ' ' );
